## Git使用说明

保存及上传修改的内容：
请依次使用：

git add . // 意味添加所有修改过的内容(包括添加或者删除的文件)
git commit -m "[写任何想要写的提交信息，用于备注]" // 意味添加一个保存点，通过命令可以简单的返回之前的保存点
git push //将本地所有的commit推到云上 **不需要每次commit立即push** 工作结束后再push即可
